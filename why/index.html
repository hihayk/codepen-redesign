I use Codepen everyday and I love it

So why I redesigned it?

Space optimization

Thick borders on code panels take unnecessary space. Also the footer bar takes vertical space which is crucial on my 13 inch laptop.

Less frequently used tools

There are a lot of buttons that I use only few times while I'm coding. Code panel options like "Tidy" are now visible only when hovering on the panel title.

Managing settings

Easier way to manage code panels and layout settings with radio style buttons. Settings are now in a side panel instead of a modal.
